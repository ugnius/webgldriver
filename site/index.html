Accelerate - Arrow key up or joystick right trigger

Brake - Arrow key down or joystick left trigger

Steer - Arrow keys left/right or joystick left stick

OK/Confirm - Enter key or joystick A

Back/Cancel - Backspace key or joystick B

Menu control - Arrow keys or joystick D-Pad

Reset Car - R key or joystick back button

Toggle help - F1 key
